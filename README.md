🚗 Vehicle Fuel & Mileage Calculator

An all-in-one mobile application built using React Native (Expo) that helps users calculate vehicle mileage, fuel cost, and driving range quickly and efficiently.

📱 Features

🔢 Mileage Calculator
Calculate your vehicle’s mileage based on distance and fuel used.

⛽ Fuel Cost Calculator
Estimate total fuel cost for a trip using distance, mileage, and fuel price.

🛣️ Range Calculator
Determine how far your vehicle can travel with the available fuel.

⚡ Simple & Clean UI
User-friendly interface for quick calculations.

🛠️ Tech Stack

Frontend: React Native (Expo)

Language: JavaScript / TypeScript

UI Components: React Native built-in components

📊 How It Works

Mileage = Distance / Fuel Used

Fuel Cost = (Distance / Mileage) × Fuel Price

Range = Fuel Available × Mileage
